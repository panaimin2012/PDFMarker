About PDF Marker

PDF Marker is a personal software to read and mark on PDF files.

License

PDF Marker is a personal software licensed with GPL (General Public License). You can use it, modify it and distribute it at no cost.

Source codes

For source codes of PDF Marker, please visit pdf-marker project at code.google.com . PDF Marker is based on muPDF (http://www.mupdf.com), a GPL library renders PDF formats.
