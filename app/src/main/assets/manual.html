PDF Marker (v0.8) User's Manual

1. How to open a PDF file

If the file is already in the recently opened list, just click the file to open it. Else use the PDF file icon from the tool panel to select the file from directory tree. To delete a file from the recently opened list, hover (long click) the item.

2. How to zoom and drag page

Use two fingers to expand or shrink the screen. When zoomed in, use one finger to drag the page.

3. What is the scissor for?

Scissor is used to cut white page edges. You can adjust page zooming and position and cut page so page edges not in screen range will not be displayed again.

4. How to turn pages

Fling on the screen wide enough to turn to next or previous page. If fling is not wide, a dialog will display asking for page number to go.

5. How to write on the page

Click the mode button in the tool panel to change it to write mode. And then you can write on the screen with a marker or pencil. In write mode, zoom and drag doesn't work. Click the mode button again to change back to hand mode which allows zooming and dragging.

6. How to change a pen (marker)

In write mode, click the stationary button, a stationary dialog will appear and allow you to change a pen (marker). You can also choose an eraser to remove your writing on the page.

8. What is a super eraser ?

A super eraser removes the intersecting strokes completely, which is recommended because it reduces the stroke in memory and disk.

9. Can I skip the page movement animation

Yes you can change the setting from preference screen.
